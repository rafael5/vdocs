# Architecture Decision Records (ADRs)

Durable, numbered records of significant architecture decisions. One file per decision,
`NNNN-kebab-title.md`, status `Proposed` / `Accepted` / `Superseded`. An ADR captures the
*context* and *consequences* of a decision so it survives the people who made it.
